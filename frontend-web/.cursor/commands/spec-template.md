# Spec 템플릿 생성 (frontend-web)

spec.md 템플릿을 생성하여 Spec-Driven Development를 시작합니다.

## 실행 절차

1. 사용자에게 기능 이름, 목표, 대상 화면/라우트를 질문한다.
2. spec.md 위치를 결정한다 (예: `docs/spec/{feature}/spec.md`).
3. `.cursor/skills/spec-template/SKILL.md` 템플릿을 참고하여 spec.md를 생성한다 (목표, 기능·비기능 요구사항, 제약, 접근성·SEO 포함).
4. 완료 후 `/plan-from-spec` 실행을 안내한다.

완료 시: "spec.md를 보완한 뒤 `/plan-from-spec`을 실행하세요."라고 안내한다.
