# Runbook Template Skill (infra)

## 구성

- **제목·대상**: 서비스/리소스·담당.
- **전제**: 접근 권한·도구·문서 링크.
- **증상**: 알람·로그·사용자 보고 요약.
- **진단 단계**: 확인할 지표·명령·로그 위치.
- **조치**: 단계별 복구·롤백 명령. 주의사항.
- **에스컬레이션**: 조건·연락처.
- **사후**: 포스트모텀·재발 방지 링크.

## 원칙

- 복사·붙여넣기 가능한 명령. 환경별 값은 placeholder. 변경 시 런북도 갱신.
