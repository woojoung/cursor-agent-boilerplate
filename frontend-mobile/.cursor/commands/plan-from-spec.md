1. spec.md 탐색·분석(목표·화면·플로우·제약).
2. lib 구조·유사 화면 참고.
3. plan.md 생성: Phase별로 화면·ViewModel·Repository 설계 → 구현 → 위젯·ViewModel 테스트.
4. 사용자 승인 대기.
