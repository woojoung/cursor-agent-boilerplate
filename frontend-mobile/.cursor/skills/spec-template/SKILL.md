# Spec Template Skill (frontend-mobile)

## spec.md 요약

- **목표**: 기능이 해결할 문제·달성 목표
- **화면·플로우**: 스크린 목록, 네비게이션, 오프라인/동기화 요구
- **요구사항**: 입력·표시·API·로컬 저장·에러·권한
- **제약**: OS 버전, 디자인 시스템, 접근성(스크린리더·폰트 크기)

작성 시 한 항목당 검증 가능하게, 플로우는 단계별로 나열.
