# Brand Guide Skill (design)

## 포함 항목

- **로고**: 사용 가능 조합·최소 크기·여백·금지 변형.
- **색상**: 프라이머리·세컨더리·기능색. 사용 비율·배경 조합.
- **타이포**: 폰트·크기·용도. 제목·본문·캡션 구분.
- **음성·톤**: 마케팅·UI 문구 톤 가이드(선택).
- **금지**: 로고 변형·색 조합·저해상도 사용 등.

문서화 시 예시·Do/Don't 포함. 디자인 시스템 토큰과 연결.
