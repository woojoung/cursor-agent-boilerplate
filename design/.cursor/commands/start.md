# 시작 (역질문 모드) — 디자인

사용자가 **뭘 해야 할지 모를 때** 옵션을 제시하고, 선택에 따라 해당 워크플로우를 안내·실행한다.

## 실행 조건

- **`/start`** 실행 시.
- 또는 "도와줘", "뭐 할 수 있어?", "디자인 관련해서" 등 **작업 내용이 불명확한 요청** 시.

## 실행 절차

1. **옵션 제시**: 이 포지션의 **PROMPT_TEMPLATES.md** "에이전트가 역질문할 때 제시할 옵션" 표를 참고해 ① 컴포넌트 규격 ② 디자인 리뷰 ③ 토큰 검증 ④ Figma/핸드오프 안내 를 보여 주고, "몇 번을 도와드릴까요?"라고 질문한다.
2. **추가 질문**: 선택에 따라 PROMPT_TEMPLATES.md의 "역질문 플로우"에 맞게 최소한의 추가 질문을 한다.
3. **실행·안내**: 해당 명령(`/component-spec`, `/design-review` 등)을 실행하거나 안내한다.

사용자가 이미 "컴포넌트 규격 만들어줘"처럼 구체적으로 요청했으면 `/start` 없이 해당 명령으로 진행한다.
